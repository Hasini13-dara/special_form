# Special Form - React Registration Form

A simple and responsive **React registration form** with routing, input validation, and a details page.

---

## Features

- Multi-page registration form using **React Router**
- Input validation for required fields
- Show/Hide password functionality
- Styled using **CSS** for a clean UI
- Details page displaying submitted information
- Fully responsive layout
